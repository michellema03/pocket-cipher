about

Ciphers are algorithms that can encrypt and decrypt messages, used in cryptography, riddle games, or just for fun. Inspired by the many overcomplicated cipher tools online, pocket cipher is a project designed to be (hopefully) simple and elegant. Just enter the information and go.


caesar cipher

what is this? First used by Julius Caesar, the Caesar cipher is a simple encryption technique where each letter in a message is shifted a fixed value along the alphabet. To use this shifter, enter the message and the amount to be shifted down the alphabet. For decrypting, if the shift value is known, simply enter its negative value to reveal the decrypted text :)

vigenère cipher

what is this? A Vigenère cipher is very similar to a Caesar cipher, but instead of a fixed shift value, the Vigenère uses a keyword instead. Each letter of the message is shifted with respect to the corresponding letter in the keyword (which repeats until the message ends). To use this shifter to encrypt, enter the message and the keyword. Decrypting coming soon!

atbash cipher

what is this? Originally used in the Hebrew alphabet, the Atbash cipher does not use shifts; instead, the Atbash uses an "inverse" alphabet to encode (and decode!) a message. A turns into Z, M turns into N, and so on. Since this process is symmetrical, decrypting and encrypting is the same.

a1-z26 cipher

what is this? The A1-Z26 cipher is a very simple substitution cipher, where each letter is replaced with its corresponding index in the alphabet, starting on 1. For decrypting, please separate numbers with a space, and words by a forward slash.
